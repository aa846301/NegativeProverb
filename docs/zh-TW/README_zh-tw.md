# NegativeProverb
***
NegativeProverb 是一個專注於負面諺語的網站，會員可以註冊並簽到。

## 技術路線
***
- ASP.NET Core Web API
- Entity Framework Core
- LINQ 語法進行資料庫查詢
- MsSQL

## 目前功能
***
- 會員註冊
  - 用戶可以註冊新帳戶，以訪問網站的所有功能。
- 簽到功能
  - 用戶可以每天簽到並獲得積分。

## 未來功能
***
- 社交分享
  - 允許用戶在社交媒體平台上分享諺語。
- 多語言支持
  - 添加對多種語言的支持，使網站可以被更廣泛的受眾訪問。
- 諺語評分和評論
  - 用戶可以對諺語進行評分和評論。
- 通知系統
  - 用戶可以接收與簽到、評論和新諺語相關的通知。
